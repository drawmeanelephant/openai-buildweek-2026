# The Agent Archive

Boris accrued a surprisingly large worker roster during Build Week. This site
will carry the curated public archive: the few detailed, evidence-backed
stories worth reading, a complete roll call, and explicitly labeled creative
artifacts.

## The rule for credit

A visible worker name is not proof of a code contribution. A prompt is not a
merged patch. A funny poster is not a release note. The archive keeps those
categories apart so the thank-you is generous without becoming fake.

## What moves here

- The complete Build Week roll call.
- Curated agent profiles with a real session, commit, pull request, or clearly
  labeled investigation/support boundary.
- Selected poster art as gallery material, never as technical evidence.
- A short collaboration retrospective after the contest.

Until that curation lands, the authoritative working archive remains in the
[Boris repository](https://github.com/drawmeanelephant/boris).
